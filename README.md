### How To Install
From a terminal, run this :

```
$ cd dotfiles
$ chmod +x scripts/install.sh
$ ./scripts/install.sh
```

And now you just need to install spf13-vim!

```
cd ~
curl http://j.mp/spf13-vim3 -L > spf13-vim.sh && sh spf13-vim.sh
```

Voila!
